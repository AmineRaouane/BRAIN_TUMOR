# Brain tumor detection

## Overview
This project focuses on detecting Tumor presence using machine learning. The dataset used for this prediction is [Brain Tumor.csv] plus a Kaggle dataset in the [Xception.ipynb].
## Technologies Used
- Python 3.x
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- streamlit
- opencv-python
- xgboost
- tensorflow
- joblib
- seaborn
